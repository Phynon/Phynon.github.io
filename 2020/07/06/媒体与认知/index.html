媒体与认知

发展历史

年代 成果
1943 神经元模型
1958 感知机
1960年之后 误差反向传播算法
1980 提出卷积神经网络
1983 提出 Hopfield 神经网络
1986 经典的误差反向传播算法
1989 发现三层神经网络的非线性拟合能力
1997 提出 LSTM 网络
1998 改进卷积神经网络
2006 提出深度置信网络
2006 提出 auto encoder

其中，监督学习的成果有卷积神经网络、深度置信网络和循环神经网络，非监督学习的成果有自动编码器 (auto encoder) 。

常见深度学习架构

Caffe 提供了在CPU和GPU上的快速卷积网络实现，使用 C++ 编写，灵活可移植，不易开发。

TensorFlow 基于张量的深度学习工具，是 Python 的域特定语言，带有丰富的算子和功能，但可读性有限。

Keras 可以使用多个后端，使用上更加简易，但相对更慢，不易开发调试。

PyTorch 基于 Torch 框架的开源 Python 机器学习库，支持自动求导，易开发调试，不易配置部署。

MXNet 兼具 Numpy 和 Keras 的特点，具有高性能和高实用性。

DGL 图神经网络工具包，有 PyTorch 和 MXNet 两种后端。
